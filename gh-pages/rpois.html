Method rpois

Description

Generates random numbers following a Poisson distribution. lambda, the (non-negative) mean can be specified as an object of class FLQuant.

Generic function

rpois(n, lambda)

Methods

signature(n=numeric, lambda=FLQuant) :
Generate a Poisson-distributed FLQuant object with 'n' iters.

Examples

   data(ple4)
   rpois(50,lambda=harvest(ple4))
